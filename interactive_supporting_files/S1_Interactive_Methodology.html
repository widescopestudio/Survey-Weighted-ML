S1 Interactive methodology — the SAFE-XAI pipeline

Survey-Aware Fairness-Constrained Evolutionary Explainable AI: an interactive walkthrough of the eleven-stage analytic pipeline applied identically across all five countries (Bangladesh development cohort; Cambodia, Myanmar, Nepal, Pakistan external validation; Indonesia assessed and excluded at Stage 3).

Click any numbered stage below, or use the flow diagram, to see what happens at that step of the pipeline and how it was validated. Use Previous/Next to step through sequentially.
This interactive file accompanies "Maternal height and fairness-aware machine learning for childhood stunting: A five-country analysis of South and Southeast Asian Demographic and Health Surveys" (submitted to Maternal & Child Nutrition). Author names are withheld from this file in keeping with the journal's double-blind peer review policy. All numerical results shown here match the values reported in the main manuscript text and the accompanying Supporting Information document (Tables S4–S22). This file requires no internet connection or external dependencies and can be opened directly in any modern web browser.
