This interactive file accompanies "Maternal height and fairness-aware machine learning for childhood stunting: A five-country analysis of South and Southeast Asian Demographic and Health Surveys" (submitted to Maternal & Child Nutrition). Author names are withheld from this file in keeping with the journal's double-blind peer review policy. Bangladesh values shown for reference are from the original development cohort (Supporting Information Table S4; AUC 0.699). Indonesia is not shown as it was excluded from modeling (anthropometric module not fielded in the 2017 survey round). This file requires no internet connection or external dependencies and can be opened directly in any modern web browser.
